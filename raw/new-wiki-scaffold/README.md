# New Wiki Scaffold

This folder contains the minimal files to copy when creating a new wiki from
workspace intake material.

## Files to Copy

- `AGENTS.md`
- `Benvenuto.md`
- `index.md`
- `log.md`
- `raw/README.md`
- `wiki/overview.md`
- `wiki/templates/README.md`
- `wiki/sources/raw/README.md`

## Usage

1. Copy these files into the new wiki folder.
2. Rename the copied content to match the new wiki subject.
3. Update links and registry entries.
