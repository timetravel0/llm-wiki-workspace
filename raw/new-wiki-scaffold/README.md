# New Wiki Scaffold

This folder contains the minimal files to copy when creating a new wiki from workspace intake material.

## Files to Copy

- [Wiki AGENTS](../../wikis/digital-thread/AGENTS.md)
- [Wiki home](../../wikis/digital-thread/Benvenuto.md)
- [Wiki index](../../wikis/digital-thread/index.md)
- [Wiki log](../../wikis/digital-thread/log.md)
- [Wiki raw README](../../wikis/digital-thread/raw/README.md)
- [Wiki overview](../../wikis/digital-thread/wiki/overview.md)
- [Wiki templates README](../../wikis/digital-thread/wiki/templates/README.md)
- [Wiki source archive README](../../wikis/digital-thread/wiki/sources/raw/README.md)

## Usage

1. Copy these files into the new wiki folder.
2. Rename the copied content to match the new wiki subject.
3. Update links and registry entries.
