# Archived Originals

This folder stores processed originals after ingest.

## Purpose

- keep the original evidence path traceable
- preserve source artifacts after they have been extracted into wiki pages
- make it clear which files were the canonical originals for a set of claims

## What Belongs Here

- original source files moved here after ingest
- raw artifacts that were assigned to this wiki and processed
- originals that should be preserved for audit or review

## What Does Not Belong Here

- drafted wiki pages
- source notes
- maintained concept, entity, tool, topic, query, or overview pages
- temporary working copies that have not been ingested yet

## Workflow

1. Ingest the source into `wiki/sources/` and update the affected maintained pages.
2. Move the processed original here only after the wiki pages capture the relevant evidence.
3. Keep filenames stable and descriptive so the provenance remains obvious.
4. Add backlinks from the source note to the archived original when useful.

## Maintenance Notes

- Do not treat this folder as a staging area.
- Do not edit the original file content here unless the workflow explicitly requires a preserved annotation copy.
- Keep this folder aligned with the archive rule in the wiki manifest and workspace AGENTS contract.
