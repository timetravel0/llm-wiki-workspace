# LLM Wiki Workspace

This is the workspace home for the multi-wiki vault.

## What Lives Here

- `raw/` for unassigned source material
- `wikis/` for independent wiki instances
- `AGENTS.md` for the global operating contract
- `index.md` for navigation
- `log.md` for workspace-level chronology
- `Decision Log.md` for durable structural decisions
- `Guida Codex Wiki.md` for practical prompts
- `LLM Wiki Operating Model.md` for the operating model

## Start Here

1. Read `AGENTS.md` for the workspace rules.
2. Read `LLM Wiki Operating Model.md` for the lifecycle model.
3. Read `Guida Codex Wiki.md` for operational prompts.
4. Use `raw/` for new material that has not yet been assigned to a wiki.
5. Register wiki instances in `wikis/index.md` when they are created.
